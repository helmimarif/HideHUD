# HideHUD

### What is this?

* This is a resource which you can use to hide HUD elements.

### Features
- Hide HUD elements.

## Download & Installation

### Using Git
```
cd resources
git clone https://github.com/helmimarif/HideHUD
```

### Manually
- Download from this.

## Installation
- Add this to your `server.cfg`:

```
start HideHUD
```
